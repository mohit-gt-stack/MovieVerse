# MovieVerse
A movie discovery web app

## 🔌 API Used

This project uses the **OMDb API (Open Movie Database API)** to fetch movie data such as:

- Movie titles
- Posters
- Release year
- Search results

### API Website
https://www.omdbapi.com/
